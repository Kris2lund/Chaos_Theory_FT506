## Overview
This project investigates nonlinear dynamical systems exhibiting chaotic behavior,
with a focus on numerical simulation, stability analysis, and bifurcation phenomena.

Such systems are highly relevant in engineering contexts including autonomous systems,
control, robotics, and cyber-physical systems, where small perturbations can lead to
large changes in system behavior.

- Bifurcation analysis

- Fractal dimension estimation (box-counting)
- Visualization and interpretation of chaotic regimes

- Phase space analysis
- Stability and sensitivity analysis

## Key takeaways
- Demonstrated how deterministic systems can exhibit unpredictable behavior
- Identified stability regions and transitions to chaos
- Quantified chaos using fractal dimensions
